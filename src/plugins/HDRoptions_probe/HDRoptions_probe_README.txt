This plugin is not intended to be used in the common sniffjoke daily activities,
when is loaded it take a command line options that specify which kind of
IP/TCP options will be used in your network.

Is required this probe, because in some network an options will be discarged and 
another will be accepted, and in other network show the opposite results.

Whenever an hacker/coder/whatever update the HDRoptions optMap, is needed updating
also this file and this plugin.
